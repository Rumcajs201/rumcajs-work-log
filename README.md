# Rumcajs Work Log

Robocza nazwa aplikacji PWA rozwijanej jako **osobisty dziennik kierowcy zawodowego**.

Aplikacja zapisuje historię pracy, pojazdów, naczep i przyczep, załadunków, rozładunków, miejsc, notatek oraz innych zdarzeń potrzebnych kierowcy do późniejszego odtworzenia przebiegu dnia.

Rejestracja czasu pracy jest jedną z funkcji, ale nie stanowi głównej koncepcji produktu.

Projekt powstaje na podstawie wieloletniego doświadczenia kierowcy zawodowego i jest rozwijany przede wszystkim pod kątem praktycznej obsługi w trasie, działania offline oraz szybkiego odnajdywania zapisów sprzed miesięcy lub lat.
